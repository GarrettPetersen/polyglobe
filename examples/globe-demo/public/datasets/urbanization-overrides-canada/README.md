# Canada Port Overrides

Manual historical port overrides for gameplay-critical Canadian Atlantic cities.

Purpose:
- Fill known gaps/misplacements in upstream urbanization sources for rail/shipping gameplay.
- Keep sparse Maritime and Newfoundland regions from being empty in early eras.

Used by:
- `scripts/build-urbanization-dominance-pruned.ts`

Current seeded cities:
- Halifax (Nova Scotia)
- Saint John (New Brunswick)
- Charlottetown (Prince Edward Island)
- St. John's (Newfoundland and Labrador)

Founding/start years used in this override:
- Halifax: 1749
- Saint John: 1785
- Charlottetown: 1764
- St. John's: 1583
